"Cha đẻ" Java nhưng bị HR đánh rớt từ vòng gửi xe khi thử xin việc Java

Published in Fun Fact
Thg 12 26, 2024
code hay ngủ

Câu chuyện dở khóc, dở cười này được chính ông chia sẻ trong một bài đăng trên facebook vào vài năm trước khi ông thử apply vào vị trí lập trình viên Java của một công ty công nghệ. Ban đầu HR từ chối nhưng rồi biết ông là "cha đẻ" của Java nên duyệt vội. Lúc phỏng vấn, HR nói rằng: “Thông thường chúng tôi không tuyển dụng lập trình viên ở độ tuổi của bạn, nhưng hoàn cảnh của bạn rất đặc biệt (vì ông ấy là cha đẻ của Java) nên bạn sẽ được quan tâm đặc biệt” =)))
